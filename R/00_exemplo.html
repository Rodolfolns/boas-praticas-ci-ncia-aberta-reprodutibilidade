2. Istalando e carregando os pacotes que vou usar

2.1 Vamos usar uma função para instalar e carregar todos os pacotes de uma vez. u seja, mesmo que você ainda não tenha determinado pacote instalado na sua maquina, a função irá fazer isso para você. A função se chama ipak. Para adicionar ou remover pacotes apenas insira ou remova do objeto packages :)

ipak <- function(pkg){
  new.pkg <- pkg[!(pkg %in% installed.packages()[, "Package"])]
  if (length(new.pkg)) 
    install.packages(new.pkg, dependencies = TRUE)
  sapply(pkg, require, character.only = TRUE)
}

packages <- c("vegan","ggplot2", "readxl","knitr","kableExtra", "htmltools", "rmarkdown", "cowplot", "here")
